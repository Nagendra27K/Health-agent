# Project Health Reporting Agent

An explainable and auditable Project Health Reporting Agent developed for the AI Engineer Intern technical assignment.

The system reads project plans, validates and normalizes imperfect data, evaluates project-health signals, determines an evidence-based Red/Amber/Green (RAG) status, explains the result in plain English, produces weekly project-health outputs, and supports monthly executive portfolio synthesis.

---

## Assignment Objective

Professional Services leadership needs reliable project-health visibility without manually chasing Project Managers every week.

This solution automates the workflow:

```text
Excel Project Plans
        ↓
Input Validation
        ↓
Schema Normalization
        ↓
Project Health Signal Extraction
        ↓
Weighted Risk Scoring
        ↓
Safeguards and Critical Overrides
        ↓
RAG Status
        ↓
Plain-English Explanation
        ↓
Weekly Project Reports
        ↓
Monthly Portfolio Synthesis
